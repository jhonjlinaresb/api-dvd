# Welcome to API-DVD


 This Api is created with love and uses the following technologies::
 - NodeJS.
 - Express.
 - Express-Validator.
 - Sequelize.
 - MySQL.
 - Body-Parser.
 - BcryptJS.
 - JSONWebToken.
 - Moment.

### Install dependencies and run:
>npm install -y

>npm start
